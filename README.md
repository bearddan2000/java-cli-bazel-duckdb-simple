# java-cli-bazel-duckdb-simple

## Description
Reads from csv and json giving field names.

## Tech stack
- java
- bazel
  - duckdb

## Docker stack
- l.gcr.io/google/bazel:latest

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
